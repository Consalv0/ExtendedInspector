# Extended Inspector

is a Unity editor extension that enhances the default Inspector experience.  
It reveals **non-serialized fields**, supports **improved attribute handling**, and makes debugging and inspection faster and clearer.

Attributes include:
- Button
- ShowInInspector
- TrackSerializedProperty (for methods and properties)
- ReadOnly
- DisableInPlayMode
- DisableInEditorMode
- HideInPlayModeAttribute
- HideInEditorMode
- InlineEditor
- InlineProperty
- FoldoutGroup
- VisibleIf
- TimeSpan
- TagField
- EditorPrefVisibility
- LayerAttribute

Custom list drawers
